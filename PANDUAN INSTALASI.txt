File Db ada di folder: cms/db/

Berikut ini cara instalasi cmsnya:

1. Buka config/config.php ubah variabel yang berhubungan dengan database dan path samakan setiap akhirnya 
   jangan nambahin / lagi serta di dalam config ada variable REQUEST, variabel REQUEST harus diubah dengan cara seperti ini:
   Misalkan alamat di local http://localhost/gundar/inaCorner maka REQUESTNYA gundar/inaCorner

2. Buka .htaccess ubah sesuai dengan lokasi foldernya 

3. Buka aksi.php terdapat variabel $path_compare ubah itu sama juga di tulisnya seperti variabel REQUEST point 1

4. Login CMS: 
   usename: administrator 
   password:administrator
